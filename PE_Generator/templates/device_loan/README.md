# Library Device Loan Management System

## Setup and Running Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Seed database:
   ```bash
   npm run seed
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```
